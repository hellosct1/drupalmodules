# drupal modules



## Drupal 7

## Présentation

Script pour détecter les modules présents `actifs` et `non  actifs`

## Installation

Placer le script à la racine du site web Drupal 7

Taper l'URL dans un navigateur

```
http://<urlSite>/ModuleScan.php
```
